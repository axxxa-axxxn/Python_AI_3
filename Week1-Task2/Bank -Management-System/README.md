# 🏦 Simple Bank Management System

A simple console-based **Bank Management System** developed using Python.  
This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts, classes, objects, functions, and basic banking operations.

---

## 📌 Features

- Create a new bank account
- Deposit money into account
- Withdraw money from account
- Check account balance
- Display all accounts
- Account search using account number
- Simple menu-driven interface

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Classes and Objects
- Functions
- Loops and Conditional Statements
