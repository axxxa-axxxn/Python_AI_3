# Personal Expense Tracker 💰

A simple Python-based Personal Expense Tracker that helps users manage their daily expenses, track spending, and monitor their remaining wallet balance.

## Features

- Add expenses
- View all expenses
- Track wallet balance
- Categorize expenses
- Input validation and error handling
- Object-Oriented Programming (OOP) implementation


## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Exception Handling


## Project Structure
Personal-Expense-Tracker/
│
├── main.py

## Classes Overview

### Expense Class
Handles expense information:

- Expense title
- Amount
- Category


### Wallet Class
Manages user wallet:

- Initial balance
- Expense deduction
- Remaining balance


### Category Class
Stores expense categories:

- Food
- Bills
- Transport
- Shopping


## How to Run

Run it from:
    python main.py
